Pry Index

cd / ls

> cd Array

> ls
Array.methods: [] try_convert
Array#methods: & * + abbrev assoc at ...

> ls         # All

> ls -m      # Methods
> ls -M      # Instance methods

> ls -g      # Globals
> ls -l      # Local vars
> ls -c      # Constants

> ls -i      # Instance vars

> ls -G xx   # Grey by regex

Shell integration

shell-mode adds dir to the prompt

pry(main)> shell-mode
pry(main):/home/x $

Commands with . are shell commands

pry(main)> .cat hello.txt
